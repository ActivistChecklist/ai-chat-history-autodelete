Unofficial · works with claude.ai

Delete old Claude chats on your schedule.

Set an age threshold, pick how often to run, and let the extension clean up your conversation list—locally in your browser. No accounts with us, no analytics, no cloud.

Install from GitHub Chrome Web Store listing coming soon.

What stays private

  • Settings and run history live in chrome.storage.local only.
  • We never see your messages, titles, or IDs—only a count of how many chats were removed per run.
  • Network use is limited to claude.ai so deletions can run in your session.

Built for control

Everything runs in Chrome. You choose when it runs and how aggressive the cutoff is.

Age threshold

Chats older than your cutoff (default 30 days) can be removed automatically or when you run a manual check.

Scheduling

Daily, weekly, monthly, or manual-only—manual is the default until you finish setup or change it.

Clear UI

A slim top bar on claude.ai when something needs your attention—progress, recent runs, or chats pending confirmation.

How it works

  1. 1

    Install & open claude.ai

    Keep a signed-in tab open when you want scheduled or manual runs—the extension uses your existing session.

  2. 2

    Configure in Settings

    Set the age threshold and schedule (or stay on manual). Optional browser notifications when action is pending.

  3. 3

    Let it run

    The next check removes conversations past your threshold—including ones that aged while no tab was open.
